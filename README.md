# Вёрстка сайта Revo Coffee

## Описание проекта

Данный проект представляет собой статическую HTML-страницу, выполненную по предоставленному макету Revo Coffee.  
Цель работы — отработка навыков семантической HTML5-вёрстки, корректного построения структуры веб-страницы и подготовки проекта к дальнейшей стилизации с использованием CSS.

В рамках текущего этапа реализована только HTML-разметка без применения CSS.

## Используемые технологии

- HTML5
