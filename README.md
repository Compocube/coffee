# Вёрстка сайта Revo Coffee

## Описание проекта

Данный проект представляет собой статическую HTML-страницу, выполненную по предоставленному макету Revo Coffee.  
Цель работы — отработка навыков семантической HTML5-вёрстки, корректного построения структуры веб-страницы и подготовки проекта к дальнейшей стилизации с использованием CSS.

В рамках текущего этапа реализована только HTML-разметка без применения CSS.

## Используемые технологии

- HTML5
- CSS

## Публикация проекта

Проект опубликован с использованием сервиса GitHub Pages и доступен по ссылке:
https://compocube.github.io/coffee/

OOCSS выбран для разделения структуры и визуального оформления, что позволяет повторно использовать стили компонентов без дублирования. В отличие от БЭМ и ACSS, OOCSS фокусируется на модульности и независимости объектов, облегчая поддержку и масштабирование кода.
